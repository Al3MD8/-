# Anime Space - فضاء الأنمي 🚀

موقع متكامل لمشاهدة وحفظ الأنمي، مبرمج للعمل محلياً باستخدام **Vite** و **Node.js**، مع ربط كامل بـ **Jikan API (MyAnimeList)** لجلب البيانات، ونظام سيرفرات مضمن (Embeds) للمشاهدة المباشرة بدون مشاكل.

## المميزات الجديدة ✨
1. **ربط API حقيقي**: جلب بيانات الأنمي (الصور، الوصف، الحلقات، التقييم) بشكل مباشر وديناميكي من Jikan API.
2. **سيرفرات متعددة (Multiple Servers)**: عند فتح أي حلقة، سيتم جلب سيرفرات المشاهدة عبر تقنية الـ iframe مع توفير حماية `no-referrer` لتخطي أي حظر.
3. **بيئة عمل احترافية (Vite)**: تشغيل محلي سريع وموثوق.
4. **صور غير قابلة للكسر**: استخدام صور MyAnimeList المباشرة مع صور احتياطية (Placeholders).

## كيفية التشغيل محلياً على جهازك 💻

أنت الآن لا تحتاج إلى الباتشر القديم (`run.bat`). الموقع يعمل كبيئة تطوير حقيقية.

### الخطوة 1: تثبيت الحزم المطلوبة
افتح موجه الأوامر (Command Prompt أو PowerShell) في هذا المجلد (`صنع انمي`) واكتب الأمر التالي لتثبيت `Vite`:
```bash
npm install
```

### الخطوة 2: تشغيل السيرفر المحلي
بعد اكتمال التثبيت، اكتب هذا الأمر لتشغيل الموقع:
```bash
npm run dev
```

### الخطوة 3: فتح الموقع
سيظهر لك رابط في الشاشة (غالباً يكون `http://localhost:5173`). انسخ الرابط وافتحه في متصفحك واستمتع بالموقع!

---

**ملاحظات تقنية:**
- الكود مصمم لجلب أحدث الأنميات تلقائياً في كل مرة تفتح فيها الموقع.
- السيرفرات المدمجة تستخدم أفضل المعايير التقنية (`allowfullscreen`, `referrerpolicy="no-referrer"`) لضمان تشغيل الفيديوهات داخل الـ `iframe`.
